Good to know

Questions worth
asking first.

Does Graft block everything on my phone?

No. You choose the apps, categories and websites. Graft only shields the distractions you put in the contract.

What counts as training?

You set the plan. Graft can verify trusted workouts and a step target for rest days using the Apple Health data you authorise.

What if I need a way out?

Graft includes timed breaks and a small lifetime emergency unlock pool. Strict Mode adds friction, but it is deliberately honest about the device owner's ultimate control.

Where can I get help?

Visit the support page for setup checks and a private route to report an issue.
